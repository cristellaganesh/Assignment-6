# Contact Form Project
This project is a contact form i built using HTML, CSS, and JavaScript.

## How it works
The form allows users to enter their name, email, and a message. JavaScript is used to validate the inputs before submission.

## Validation
- Their name must not be empty and must contain only letters
- Email must follow a valid email format
- Message must be at least 10 characters long


## Modal Interaction
When all fields are valid and the user submits the form, a modal appears displaying the entered information. The modal can be closed by clicking the close button or outside the modal.
